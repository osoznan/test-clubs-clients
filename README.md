**Как развернуть**

Привычное для yii2-advanced:
```
php yii init
```

Миграции:
```
php yii migrate
```

Для генерации (и перегенерации) тестовых данных:
```
php yii generate-test-data
```
Для входа в аккаунт следующие данные:
```
логин: admin
пароль: 12345678
```
